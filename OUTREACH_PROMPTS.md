## Rate Limit Reminder

- Max 1-2 DMs per minute

---

## Quick Reference - MCP Tools Available

- `search_public_chats(query)` - Find groups
- `subscribe_public_channel(channel)` - Join public groups
- `get_admins(chat_id)` - Get admin list
- `send_message(chat_id, message)` - Send DM
- `list_chats()` - See all your chats
- `get_messages(chat_id)` - Read messages
- `search_messages(chat_id, query)` - Search in chat

## STYLE NOTES FOR ALL IMAGES

- **Pixel art / 16-bit aesthetic** - Match the existing Pepe images
- **No anti-aliasing** - Clean pixel edges
- **Limited color palette** - Max 8-10 colors per image
- **Dark backgrounds** - #0a0a0f or pure black
- **Green glow effects** - #00ff88 with bloom/glow
- **Pepe frog as mascot** - Consistent character design
- **"Sealed" visual motif** - Wax seals, stamps, checkmarks
- **Crypto/degen energy** - Not corporate, not boring
